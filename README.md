# Vue.js Shop SPA

Интернет-магазин на Vue 3 с корзиной, авторизацией и оформлением заказов. Проект выполнен в рамках 4 задания практической работы.

## Функциональность

- Регистрация и вход пользователей (JWT-токены)
- Просмотр каталога товаров с изображениями
- Добавление товаров в корзину
- Просмотр корзины с группировкой товаров и подсчётом количества
- Изменение количества товаров в корзине
- Оформление заказа
- Просмотр списка оформленных заказов
- Защита маршрутов (неавторизованные пользователи не могут попасть в корзину и на страницу заказов)

## Технологии

- **Vue 3** — основной фреймворк
- **Vuex** — управление состоянием
- **Vue Router** — маршрутизация
- **Fetch API** — взаимодействие с бэкендом
- **CSS3** — стилизация

## Требования

- Node.js
- npm

## Структура проекта
src/
├── assets/            # статические ресурсы (изображения, шрифты)
├── components/        # переиспользуемые компоненты (ProductCard, Login, Register и т.д.)
├── views/             # компоненты-страницы (HomeView, CartView, OrdersView)
├── router/            # настройки маршрутизации
├── store/             # Vuex store (состояние, мутации, действия)
├── utils/             # вспомогательные модули (api.js для запросов)
├── App.vue            # корневой компонент
└── main.js            # точка входа
